# Midnight Firefox Theme
A better dark theme for Firefox.

## Install
 - [Install at the Firefox Add-ons Store.](https://addons.mozilla.org/en-US/firefox/addon/midnight-dark-theme/)
 - Download from the "Releases" tab on GitHub.
